**Focus:** HTML, CSS (Sass), JavaScript, TypeScript, Testing, Bundlers (Webpack, Vite), RESTful APIs, Asynchronous programming.

This course is designed for beginners and students who want to gain strong, practical experience in JavaScript and modern front-end development.  
It follows a **self-study format**, supported by **webinars**, **community discussions**, and a structured system of **assignments**, **cross-check reviews**, and **mentor evaluations**.

Throughout the course, you build real projects, solve algorithmic tasks, learn HTML/CSS/JS from the ground up, and gradually move toward more advanced front-end topics.

The final stage is **7-week team development sprint**, simulating real work in a front-end team.
